Task: Selecionar Tarefa Diária
O Desenvolvedor com a aprovação do Scrum Master do Produto seleciona uma das tarefas que não foram iniciadas.
Purpose
Selecionar uma tarefa para conclusão do Sprint.
Main Description

O Desenvolvedor deve selecionar uma das tarefas em aberto para que possa iniciar a execução. As tarefas em aberto podem ser consultadas através do Redmine.

O Scrum Master do Produto deve aprovar a escolha e associar essa tarefa para que possa ser executada pelo Desenvolvedor.

Properties
Multiple Occurrences
Event Driven
Ongoing
Optional
Planned
Repeatable
